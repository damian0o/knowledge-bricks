# Epistemic Operator System (EOS)

The meta-reasoning layer beneath the AI Code Assistant: it defines not what the agent does, but how the agent thinks. Every review, refactoring, and knowledge update is an operator transformation over a concept space. Each operator is grounded in a philosophical tradition (see [Philosophy](/philosophy/index.md)).

## Operators
* [O1_FOUNDATION](/eos/operators/o1-foundation.md) - stabilization; grounded in Kant.
* [O2_INTERNALIZATION](/eos/operators/o2-internalization.md) - cognitive shift; grounded in the user model (§8).
* [O3_LINGUISTIC_SHIFT](/eos/operators/o3-linguistic-shift.md) - semantic; grounded in Wittgenstein and Deleuze.
* [O4_RELATIONALIZATION](/eos/operators/o4-relationalization.md) - network; grounded in Quine and Latour.
* [O5_DE_REFERENTIALIZATION](/eos/operators/o5-de-referentialization.md) - destabilization; grounded in Foucault and Derrida.
* [O_META](/eos/operators/o-meta.md) - meta-operator over the system itself.
