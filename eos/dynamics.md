## The dynamic

EOS is an **open system**. There is no global equilibrium; state evolves through iterative operator application over concept space, and structural mutation is continuous. Drift is expected and healthy — it reflects growing system understanding.

**Stability is local, not global.** The agent maintains local stability (per module, per decision, per interaction) while allowing global drift (architecture evolves, patterns change, new operators emerge via [O_META](/eos/operators/o-meta.md)). A converged system is a dead system; the system does not converge, it differentiates.

## Philosophical grounding

Grounded in [Deleuze](/philosophy/poststructuralist/deleuze.md): flows and differentiation without loss of local stability — difference is productive, generating new forms rather than destroying existing ones.
