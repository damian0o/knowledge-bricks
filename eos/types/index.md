# EOS Core Types

The three core types of the Epistemic Operator System.

## Types
* [Concept](/eos/types/concept.md) - an atomic or composite semantic unit that operators transform.
* [Operator](/eos/types/operator.md) - a function O: Concept × Context → Concept.
* [Configuration](/eos/types/configuration.md) - a composition of operators forming a reasoning chain.
