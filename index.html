knowledge-bricks — the EOS Knowledge Base

A portable OKF v0.1 knowledge base encoding the Epistemic Operator System (EOS) — the meta-reasoning layer of the AI Code Assistant (ACA v5) — together with the philosophical foundations that ground each operator. Every file is one concept; links between files form a graph spanning philosophy ↔ EOS ↔ engineering practice.

The system

  • EOS — Epistemic Operator System - operators, axioms, types, and system dynamics; how the agent thinks

The foundations

  • Philosophy — conceptions of reality - the graph of thinkers from Plato’s cave to quantum reality that grounds EOS

How to read this bundle

  • Each concept file carries YAML frontmatter with a type; the body states the idea and links related concepts.
  • A link asserts a relationship; the prose around it names the kind (influenced / reacts-against / refines / opposes / synthesizes / grounds).
  • The 7 operator-grounding thinkers — Kant, Quine, Wittgenstein, Deleuze, Foucault, Derrida, Latour — link forward to the EOS operator they ground; each operator links back to them.
  • The “next step for philosophy to resolve” is deliberately left unwritten — the empty socket the graph builds toward.
